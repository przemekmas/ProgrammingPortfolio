About Me


I developed an interest in building computers at an early age of 13, which was when I have built my first gaming computer. I have always liked to be able to build computers by myself and I keep up to date with the latest PC hardware to this day. However, I developed an interest in programming after I joined university. The first programming language that I learned was Java, which was my first object oriented programming language that I picked up. Later throughout my university I have picked up C#. After graduating with a first class degree from the University of Bedfordshire, I have secured a job as a Junior Software Developer. I have worked for about 1.5 years before being promoted to an Intermediate Software Developer. I have expereince with desktop based applications written in C#, WPF, and WinForms. I also have expereince with writing android applications and the Xamarin framework. After 4 years of developing desktop and mobile applications, I have moved on to web applications and have secured a job as a Software Engineer. I have decided to move on from desktop applications as I always liked to work on web applications in my spare time. I have also made the decision to start working on web based applications as I believe that we moving on from desktop applications to web applications as they are easier to access and maintain.
